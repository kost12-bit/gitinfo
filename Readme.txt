Клонирование проектв в папку nameProject:
git clone https://... nameProject

Добавить файлы в комит:
git add .

Запустить коммит:
git commit -m "Коментарий коммита"

Отправить коммит:
git push

Принять изминения:
git pull

Авторство коммитов:
git config --global user.email "ваш емайл"
git config --global user.name "Ваше имя"
